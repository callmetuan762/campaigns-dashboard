```sql
FROM ad_metrics
WHERE ad_set_id = '' AND ad_id = ''
  AND spend > 0;
```

MMM data load SQL (use meta_form_submit_deposit — NOT meta_purchases_7dclick):
```sql
SELECT date,
       SUM(spend)                    AS daily_spend,
       SUM(meta_form_submit_deposit) AS daily_deposits
FROM ad_metrics
WHERE ad_set_id = '' AND ad_id = ''
  AND spend > 0
GROUP BY date
ORDER BY date ASC;
```

**`build_mmm_telegram_message(result: MMMResult, week_label: str, deposit_value_usd: float = 0.0) -> str`** — pure function. Build message per D-08 format. Include "⚠ Directional only" line only when `result.maturity_label == "directional_only"`. When `deposit_value_usd == 0.0`: ROAS line = `f"Meta generated {result.incremental_roas_per_1k:.1f} deposits per $1000 spend."`. When `deposit_value_usd > 0.0` and `result.incremental_roas_per_1k` is not None: ROAS line = `f"Incremental ROAS: {result.incremental_roas_per_1k:.1f}x (every $1 of Meta spend generated ${result.incremental_roas_per_1k:.1f} in deposit value)."`. When `result.incremental_roas_per_1k` is None: omit ROAS line entirely.

**`async def run_mmm_weekly_job() -> None:`** — zero-arg function (APScheduler requirement).
1. Assert `_bot and _db and _settings` — raise RuntimeError if not set.
2. Week count: `rows = await _db.fetch_all("SELECT COUNT(DISTINCT strftime('%Y-%W', date)) AS weeks FROM ad_metrics WHERE ad_set_id='' AND ad_id='' AND spend > 0")`. Extract `weeks = rows[0]["weeks"] if rows else 0`.
3. If `weeks < 4`: `logger.info("mmm_job_skipped_insufficient_weeks", weeks=weeks)` and return.
4. Load data: `data_rows = await _db.fetch_all("SELECT date, SUM(spend) AS daily_spend, SUM(meta_form_submit_deposit) AS daily_deposits FROM ad_metrics WHERE ad_set_id='' AND ad_id='' AND spend > 0 GROUP BY date ORDER BY date ASC")`. Extract `spend = np.array([r["daily_spend"] for r in data_rows], dtype=float)` and `deposits = np.array([r["daily_deposits"] for r in data_rows], dtype=float)`.
5. Run model in thread: `result = await asyncio.to_thread(fit_mmm, spend, deposits, _settings.deposit_value_usd, date.today().isoformat(), weeks)`.
6. If `result is None`: `logger.warning("mmm_fit_failed", weeks=weeks)` and return.
7. Persist: `await _db.upsert_mmm_result(result)`.
8. Build message: `week_label = date.today().isoformat()`. Call `build_mmm_telegram_message(result, week_label, _settings.deposit_value_usd)`.
9. Send to all chat IDs: `for chat_id in _settings.telegram_allowed_chat_ids: await _bot.send_message(chat_id=chat_id, text=text)` in a try/except logging send errors without re-raising.
