```python
_bot = None
_db = None
_settings = None

_RECENT_FAILURES_SQL = """
    SELECT status FROM ingestion_log
    WHERE source = :source
    ORDER BY started_at DESC
    LIMIT :limit
"""

def register_job_resources(bot, db, settings) -> None:
    global _bot, _db, _settings
    _bot = bot
    _db = db
    _settings = settings
    logger.info("ingest_resources_registered")

async def _check_circuit_breaker(db, source: str, threshold: int = 3) -> bool:
    rows = await db.fetch_all(_RECENT_FAILURES_SQL, {"source": source, "limit": threshold})
    if len(rows) < threshold:
        return False
    return all(r.get("status") == "failed" for r in rows)

async def meta_ingest_job() -> None:
    if _bot is None or _db is None or _settings is None:
        logger.error("ingest_job_resources_not_registered")
        return
    await _run_meta_ingest(_bot, _db, _settings)
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GA4 API → ingest job | Data from Google API; campaign names and landing page paths are externally controlled strings |
| service account JSON file → client | Credentials loaded from disk; path from settings only — never from user input |
| ingest job → Telegram (circuit breaker) | Error message snippet sent to Telegram when circuit breaks |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Information Disclosure | ga4_service_account_json path/contents | mitigate | Path loaded from settings env var; file contents handled by Google client library only; never logged or passed to user-visible output |
| T-03-02-02 | Tampering | Campaign name / landing page in GA4 API response | mitigate | All strings from GA4 API passed to html.escape() at report render time (builder.py); stored as raw strings in SQLite |
| T-03-02-03 | Elevation of Privilege | GA4 Data API access scope | accept | BetaAnalyticsDataClient uses analytics.readonly scope internally; Viewer-only IAM role enforced at GA4 Admin level; read-only by design |
| T-03-02-04 | Denial of Service | GA4 quota exhaustion | mitigate | 6-hour cache check prevents >1 API call per 6h; return_property_quota=True enables quota monitoring; single daily schedule at 01:00 |
| T-03-02-05 | Tampering | Circuit breaker Telegram message (error string) | mitigate | `html.escape(str(exc)[:200])` — escaped and truncated before send; no raw error passed to ParseMode.HTML context |
</threat_model>
